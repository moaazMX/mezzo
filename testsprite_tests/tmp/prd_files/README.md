# مشروع MX (Mezzo) - نظام إدارة وطلب الوجبات (Gaming Theme)

هذا المشروع عبارة عن تطبيق ويب متكامل لطلب الوجبات وإدارة المطاعم، يتميز بتصميم فريد مستوحى من عالم الألعاب (Gaming)، مما يوفر تجربة ممتعة وتفاعلية للمستخدمين.

## 🚀 التقنيات المستخدمة
- **Frontend**: React.js مع TypeScript.
- **Bundler**: Vite لضمان سرعة الأداء والتطوير.
- **Styling**: Tailwind CSS مع دعم كامل للوضع الداكن (Dark Mode) والاتجاه من اليمين لليسار (RTL).
- **Backend**: Supabase (قاعدة بيانات PostgreSQL، المصادقة، والتخزين).
- **Maps**: Leaflet لإدارة مناطق التوصيل وتحديد مواقع العملاء بدقة.
- **Icons**: Lucide React.
- **Fonts**: Press Start 2P (لللمسة الكلاسيكية للألعاب).

## ✨ المميزات الرئيسية

### 1. الطابع الخاص بالألعاب (Gaming Theme)
- استخدام مصطلحات من عالم الألعاب في أسماء الوجبات (مثل: Noob Burger, Double Damage, Sniper Chicken, XP Boost, Mana & Potions).
- واجهة مستخدم بتصميم داكن وألوان فسفورية جذابة.
- تأثيرات صوتية وحركية مستوحاة من الألعاب (Retro Gaming Style).

### 2. تجربة المستخدم (UX/UI)
- تصميم عصري وسريع الاستجابة (Responsive) يتناسب مع كافة الأجهزة.
- تأثيرات حركية متطورة عند إضافة المنتجات للسلة (Flying items animation).
- دعم كامل للغتين العربية والإنجليزية مع تبديل لحظي.
- نظام تبديل الحسابات (Multi-Account) يتيح للعميل حفظ أكثر من حساب (Profile) على نفس الجهاز مع سهولة التبديل بينهم.

### 3. نظام الطلبات والتوصيل المتطور
- عرض المنتجات حسب الفئات مع أيقونات تعبيرية.
- سلة مشتريات ذكية تتيح تعديل الكميات والخيارات بسهولة.
- **نظام توصيل ذكي**: يعتمد على الخرائط التفاعلية والطبقات (Polygon Layers) لتحديد مناطق التوصيل (Delivery Zones) وحساب تكلفة التوصيل بناءً على الموقع الدقيق للمستخدم.
- متابعة حالة الطلب لحظياً مع تحديثات تلقائية (قيد المعاينة، قيد التحضير، في الطريق، تم الوصول).
- دعم طرق الدفع المختلفة: نقداً (Cash) أو تحويل فوري (Instant Transfer).

### 4. الأمان والخصوصية
- **وضع المشغل (Operator Mode)**: محمي بكلمة مرور لإدارة الطلبات والبيانات.
- **حماية الحساب**: إمكانية إعداد كلمة مرور اختيارية لرقم الهاتف لمنع التلاعب بالطلبات من أجهزة أخرى.
- **بصمة الجهاز (Device Fingerprint)**: تقنية متقدمة للتعرف على الأجهزة لضمان نزاهة استخدام الكوبونات وتتبع سجل الطلبات.

### 5. نظام الأكواد السرية (Cheat Codes)
- نظام فريد يتيح للعملاء كتابة "شفرات" مباشرة على الشاشة بلوحة المفاتيح لفتح كوبونات خصم مخفية.
- لوحة تحكم لإدارة هذه الأكواد ونسب الخصم وفترة صلاحيتها.

### 6. لوحة التحكم (Operator Dashboard)
- واجهة احترافية للموظفين لمتابعة الطلبات الجديدة وإدارتها.
- نظام تنبيهات (Toasts) متطور.
- عرض تفاصيل العميل وموقعه على الخريطة بدقة لتسهيل عملية التوصيل.

## 📁 هيكل المشروع
- `src/components`: المكونات الأساسية (Cart, Checkout, CustomerProfile, InteractiveMap, CheatCodeInput, etc).
- `src/operator`: واجهات خاصة بلوحة تحكم المشغل.
- `src/contexts`: إدارة اللغة، السمة، والمصادقة.
- `src/lib`: الربط مع Supabase ومنطق الخرائط والبصمة الرقمية.
- `supabase/`: ملفات التهيئة والسياسات الأمنية للبيانات.

## 🛠️ المتطلبات والتشغيل
- **Node.js**: إصدار 18 أو أحدث.
- **Supabase**: حساب مفعل وقاعدة بيانات مهيأة بالملفات المرفقة (`.sql`).
